## Playdata Chat

<br>

NEXT13을 이용한 실시간 채팅 웹 앱.

#### 주요 기능
```
회원가입/로그인/소셜로그인(깃헙, 구글)
이용자간 1:1 또는 그룹 채팅
Cloudinary를 이용한 사진 전송
```
